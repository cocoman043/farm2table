## How to use

1. Clone the project
```bash
git clone https://github.com/cocoman043/farm2table.git
```
2. Start the frontend
```bash
cd farm2table/client
npm i
npm start
```
3. Start the backend
```bash
cd farm2table/server
npm i
npm start
```
